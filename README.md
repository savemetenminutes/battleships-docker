## Battleships

### Installation
```$bash
git clone --recurse-submodules https://github.com/savemetenminutes/battleships-docker.git savemetenminutes-battleships-docker
cd savemetenminutes-battleships-docker
docker-compose up
```

### Usage
Open a browser and navigate to http://127.0.0.1:45083
